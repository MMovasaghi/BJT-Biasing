## Circuits

For all the circuits we must get `Î²` , `V_BE` , `V_Sat` .

By default they are assign such this value:

` V_Sat = 0.2 `(`V_Sat` is `V_CE` in the **Saturation** mode)

` V_BE = 0.7 `
